    OPERADORES
        * Aritméticos
            Usado pra fazer cálculos

            Soma 5 + 2                          -> 7
            Subtração 5 - 2                     -> 3
            Multiplicação 5 * 2                 -> 10
            Divisão real 5 / 2                  -> 2.5
            Resto da divisão inteira 5 % 2      -> 1
            Potência 5 ** 2                     -> 25

            Ordem de precedência
            1º ()
            2° **
            3° * / %
            4° + -

        * Atribuição
            AUTO-ATRIBUIÇÃO
            var n = 3
            n = n + 4               -> 7
            n = n - 5               -> 2
            n = n * 4               -> 8
            n = n / 2               -> 4
            n = n ** 2              -> 16
            n = n % 5               -> 1

            SIMPLIFICANDO
            var n = 3
            n += 4               -> 7
            n -= 5               -> 2
            n *= 4               -> 8
            n /= 2               -> 4
            n **= 2              -> 16
            n %= 5               -> 1

            POS INCREMENTO e POS DECREMENTO

            var n = 5
            n++ (somar 1) -> 6
            n-- (subtrair 1) -> 5

            PRE INCREMENTO e PRE DECREMENTO
            var n = 5
            ++n (somar 1) -> 6
            --n (subtrair 1) -> 5

        * Relacionais
            Maior            5 > 2       -> true
            Menor            7 < 4       -> false
            Maior ou igual   8 >= 8      -> true
            Menor ou igual   9 <= 7      -> false
            Igual            5 == 5      -> true
            Diferente        4 != 4      -> false

            # OPERADORES DE IDENTIDADE
            Igual                5 == 5      -> true
            Igual                5 == '5'    -> true
            Idêntico             5 === '5'   -> false 
            Idêntico             5 === 5     -> true
            O Idêntico compara o tipo e o valor
            Desigual restrito    5 !== '5'     -> true
            O desigual restrito também compara tipo e valor

        * Lógicos
            Não         !       -> Negação
            E           &&      -> Conjunção
            OU          ||      -> Disjunção


            # NEGAÇÂO
            True -> false
            False -> true

            # CONJUNÇÃO
            True        &&      True        -> True
            True        &&      False       -> False
            False       &&      True        -> False
            False       &&      False       -> False

            # DISJUNÇÃO
            True        ||      True        -> True
            True        ||      False       -> True
            False       ||      True        -> True
            False       ||      False       -> False

            # Ordem de precedência
            1º !
            2º &&
            3º ||

    ACESSANDO UM ELEMENTO

        * por Marca
            getElementsByTagName('p')[1] -> da pra selecionar mais de 1 tag <p></p> o [] escolhe qual parágrafo que quer fazer a alguma coisa
            var corpo = window.document.body -> ta acessando o body da pagina
            var p1 = window.document.getElementsByTagName('p')[0] -> pegou o primeiro parágrafo
            corpo.style.background = 'black' -> mudou a cor de fundo do body
            p1.style.color = 'blue' -> mudou a cor do primeiro parágrafo
            document.write(p1.innerText) -> pega o que está escrito no parágrafo sem formatação
            document.write(p1.innerHTML) -> pega o que está escrito no parágrafo, com as formatações das tags

        * por ID
            getElementById()
            Pega pelo Id da tag

            <div id="msg"></div>
            var d = window.document.getElementById('msg')
            d.style.background = 'green' -> muda a cor do fundo
            d.innerText = 'Estou aguardando...' -> muda para esse texto
            window.document.getElementById('msg').innerText = 'Olá' -> pode ser feito desse jeito tbm

        * por Nome
            getElementsByName()
            pega vários elementos então tem que referenciar qual o elemento q irá ser alterado

            <div name="msg"></div>
            var d = window.document.getElementsByName('msg')[0]
            d.innerText = 'Olá' -> muda para esse texto

        * por Classe
            getElementsByClassName()
            pega pela classe e pode pegar vários elementos

            <div class="msg"></div>
            var d = window.document.getElementsByClassName('msg')
            d.innerText = 'Olá' -> muda para esse texto
            window.document.getElementsByClassName('msg').innerText = 'Olá' -> pode ser feito desse jeito tbm

        * por Seletor
        querySelector()
        querySelectorAll()

        <div id="msg"></div>

        se for id #, se for classe .
        var d = window.document.querySelector('div#msg')
        d.style.color = 'blue'

    # EXEMPLO
        <div id="area">Interaja....</div>

        var a = window.document.getElementById('area')

        Colocar esses addEventListener em vez de colocar na tag
        a.addEventListener('click', clicar)
        a.addEventListener('mouseenter', entrar)
        a.addEventListener('mouseout', sair)

        function clicar() {
            a.innerText = 'Clicou!'
            a.style.background = 'red'
        }
        function entrar() {
            a.innerText = 'Entrou!'
        }
        function sair() {
            a.innerText = 'Saiu!'
            a.style.background = 'green'
        }
