Condições

        
            if(condicao) {
                True
            } else {
                False
            }

Tipos de Condiçao

Condição Simples

Se a condição for falsa, nada irá acontecer.

        
            if(condicao) {
                True
            }

Condição Composta

Nessa situação se for verdadeira vai acontecer alguma coisa e se for falsa vai acontecer outra coisa.

        
            if(condicao) {
                True
            } else {
                False
            }

Condições Aninhadas

Aqui uma condicao vai aninhando em outra, nao existe so verda ou falso, ha mais possibilidades

        
            if (condicao 1){
                bloco1
            } else if (condicao 2) {
                    bloco 2
                } else {
                    bloco 3
                }
            }
